# Strings

## Overview
This directory contains solutions to common string manipulation problems
frequently asked in technical interviews.

The problems focus on efficient string traversal, character frequency tracking,
and sliding window techniques.

## Concepts Covered
- Character frequency counting
- Sliding window technique
- Hash-based lookups
- String traversal and indexing

## Problems Included
- Valid Anagram
- Longest Substring Without Repeating Characters

## Key Learnings
- Efficient handling of string-based data
- Avoiding unnecessary nested loops
- Writing clean, readable logic for string problems
